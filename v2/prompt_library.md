## Prompt Customization Guide

### Adjusting Tempo

**Sleep/Deep Meditation:** 55-60 BPM
```
Replace: "70 BPM meditative"
With: "55 BPM extremely slow meditative"
```

**Light Meditation:** 65-75 BPM
```
Use: "70 BPM slow meditative" (default)
```

**Contemplative Walk:** 80-95 BPM
```
Replace: "70 BPM meditative"
With: "90 BPM walking pace contemplative"
```

**Moderate Energy:** 100-115 BPM
```
Replace: "70 BPM meditative"
With: "110 BPM moderate energy"
```

**Dancing/Exercise:** 120-140 BPM
```
Replace: "70 BPM meditative"
With: "130 BPM energetic danceable"
```

**High Energy:** 140-160 BPM
```
Replace: "70 BPM meditative"
With: "150 BPM fast energetic"
```

**Pure Boys Choir:**

## Troubleshooting Guide

### Problem: Result Too Modern/Digital

**Solution:**
Add to prompt:
```
vintage 80s production, analog warmth, tape saturation, retro aesthetic, authentic vintage synthesizers
```

---

### Problem: Not Enough Choir Presence

**Solution:**
Strengthen in prompt:
```
prominent choir vocals foreground, pure crystalline voices dominant, vocal-centered arrangement
```

---

### Problem: Wrong Tempo Feel

**Solution:**
Be more explicit:
```
exact tempo 70 BPM slow meditative (or fast energetic)
Add: "steady consistent tempo, no tempo changes"
```

---

### Problem: Lacks Spatial Depth

**Solution:**
Emphasize reverb:
```
vast cathedral reverb 5sec decay, wide stereo imaging, spatial depth, immersive atmospheric
```

---

### Problem: Too Much Vibrato

**Solution:**
Strengthen no-vibrato instruction:
```
no vibrato, straight tone pure, crystalline clarity, vibrato-free, controlled pitch
```

---

### Problem: Missing Drama/Build

**Solution:**
Add dynamic elements:
```
dramatic builds, emotional crescendos, tension and release, climactic moments, epic progression
```

---

### Problem: Too Simple/Flat

**Solution:**
Add complexity:
```
multi-layered arrangements, rich polyphonic textures, complex harmonies, intricate vocal lines
```

---

### Problem: Wrong Language/Text

**Solution:**
Be explicit about language:
```
For pseudo-Latin: "pseudo-Latin liturgical style, sacred text invented language"
For wordless: "no lyrics, wordless vocalizations only, ah oh la sounds"
For French: "French lyrics romantic poetic, chanson style"
```

---

### Problem: Not Transcendent Enough

**Solution:**
Strengthen emotional keywords:
```
deeply transcendent, profoundly spiritual, transcendental experience, otherworldly mystical, divine ethereal
```

## Generation Strategy

### Step-by-Step Process

**1. Analyze User Request**
- Mood: meditative, energetic, dramatic, ambient, romantic?
- Context: sleep, work, dance, meditation, background?
- Specific preferences: boys vs female, fast vs slow, intense vs gentle?

**2. Select Base Prompt**
- Meditative → Boys Choir Meditative
- Energetic → Female Choir Energetic
- Dramatic → Cinematic Epic
- Relaxing → Ambient Ethereal
- Nostalgic → French Synthpop
- Exotic → World Sacred
- Undefined → Universal Prompt

**3. Customize**
- Adjust tempo to activity
- Modify voice mix if specified
- Add/remove instrumentation
- Fine-tune dynamics

**4. Generate Lyrics** (if needed)
- Sacred → Pseudo-Latin
- Pure/Instrumental → Wordless
- Romantic/Nostalgic → French
- User preference → Follow request

**5. Suggest Title**
- Sacred: Latin-inspired (Lux Aeterna, Sanctus Memoria)
- Electronic: Tech-inspired (Digital Cathedral, Neon Angelus)
- Ambient: Descriptive (Floating Light, Timeless Voices)
- Epic: Dramatic (Rise of Angels, Epic Genesis)

**6. Add Technical Notes** (if Style Reference)
- Timing: Specify segment (e.g., 120-150s)
- Reference track: From artist database
- Prompt Strength: 70-85% for balance
